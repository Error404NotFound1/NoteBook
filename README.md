# NoteBook

欢迎来到我的笔记仓库。

这里记录一些日常想法、学习过程、灵感碎片，以及那些暂时还没来得及整理清楚的内容。  
它不是一个完整的作品集，更像是一个慢慢生长的思考空间。

## 关于这里

这个仓库主要用 Obsidian 维护，内容会随着时间不断更新。  
有些笔记很短，有些笔记可能只是一个开头；  
有些内容是给自己看的，有些内容也许以后会变成一篇完整的文章。

## 这里可能会有

- 日常记录
- 学习笔记
- 技术整理
- 想法草稿
- 阅读摘录
- 还没想好名字的东西

## 我怎么整理它

- 用 [[链接]] 把相关内容连接起来
- 用标签记录主题
- 把零散的想法慢慢整理成可复用的知识
- 保持足够简单，不让整理本身变成负担

## 仓库里的一些约定

- `.obsidian/` 用来存放本地配置
- `.copilot/` 用来存放辅助工具相关文件
- `.DS_Store` 这类系统文件会被忽略
- 真正有价值的，是那些 `.md` 笔记本身

## 结尾

如果你也喜欢这种边写边整理的方式，那我们大概会很合拍。  
这里不会一直完美，但会一直更新。
